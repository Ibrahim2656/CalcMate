## How to Use

1. Open the deployed project in your browser.  
2. Use the calculator with mouse or keyboard:  
   - Numbers: `0-9`  
   - Operators: `+ - * /`  
   - Enter: Calculate result  
   - Backspace: Delete last character  
   - C: Clear display  
   - Theme toggle button: Switch between dark/light mode
